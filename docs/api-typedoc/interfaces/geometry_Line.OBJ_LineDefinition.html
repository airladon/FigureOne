    Interface OBJ_LineDefinition

    Line definition object.

    Combinations that can be used are:

    • p1, p2, ends
    • p1, length, angle, ends (for 2D lines)
    • p1, length, phi, theta, ends (for 3D lines)
    • p1, length, direction, ends
