Sine Waves In Time and Space

When the sine function is plotted on a Cartesian axis it “looks like” a wave.

Move the circle radius to draw the sine wave.

A form of this wave is found in many physical phenomena, but it often needs to be compressed, stretched or shifted to properly model the phenomena.

It is therefore useful to know how to change the equation to achieve these transformations.

Vertical Scaling

To stretch or compress a sine function in \(y\), the function can be scaled with a constant \(A\).

\[y=A\sin\left(\theta\right)\]

Drag the trace up and down to see how the trace and equation are related.

Horizontal Scaling

The sine function repeats every time the input value \(\theta\) changes by \(2\pi\).

What happens if we multiply the input value by \(2\pi\)?

\[y=\sin(2\pi\theta)\]

Now, every time the value \(\theta\) changes by 1, the input to the sine function changes by \(2\pi\).

Instead of 1, if we want the function to repeat every change \(r\) we can:

\[y=\sin\left(2\pi{\theta \over r}\right)\]

Now every time \(\theta\) changes by \(r\), the input to the sine function changes by \(2\pi\).

Let's rewrite this as:

\[\bbox[20px, border: 2px solid red]{y=\sin\left({2\pi \over r}\theta\right)} \ \ \ \ \ \ \ \ \ (1)\]

Drag the below plot left and right to see how the equation changes. As \(2\pi\) is approximately 6.3, then dragging it to show 6.3 means the \(2\pi\) term cancels and we are left with \(\sin\left(\theta\right)\).

Sine Waves in Time

On a time axis, equation (1) defines the period of time over which the sine function repeats. As the period is often written as \(T\) we have:

\[y=\sin\left({2\pi \over T}t\right)\]

If the sine function repeats every \(T\) seconds, then the frequency it repeats is:

\[f={1 \over T}\]

Therefore we have:

\[y=\sin\left(2\pi f t\right)\]

\(2\pi f\) is the number of times \(2\pi\) is cycled through per second, and therefore is often called the angular frequency \(\omega\).

\[\bbox[20px, border: 2px solid red]{y=\sin\left(\omega t\right)}\]

Sine Waves in Space

On a space axis (\(x\)), equation (1) defines the distance over which the sine function repeats. When dealing with physical lengths, this is usually called the wavelength \(\lambda\).

\[y=\sin\left({2\pi \over \lambda}x\right)\]

The \({2\pi \over \lambda}\) term describes how many radians per unit distance the sine wave progresses through, and is often called the wave number \(k\):

\[\bbox[20px, border: 2px solid red]{y=\sin\left(kx\right)}\]

Both \(k\) and \(\omega\) serve the same function for space and time respectively. Both give the number of times the sine function repeats per unit length or unit time.
