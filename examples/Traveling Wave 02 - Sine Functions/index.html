Sine Waves

When the sine function is plotted on a Cartesian axis it “looks like” a wave.

This form of this wave is found in many physical phenomena, but it often needs to be compressed, stretched or shifted to properly model the phenomena. It is therefore useful to know how to change the equation to achieve these transformations.

Horizontal Scaling

The sine function repeats every time the input value \(\theta\) changes by \(2\pi\).

What happens if we multiply the input value by \(2\pi\)?

\[y=\sin(2\pi\theta)\]

Now, every time the value \(\theta\) changes by 1, the input to the sine function changes by \(2\pi\).

Instead of 1, if we want the function to repeat every change \(r\) we can:

\[y=\sin\left(2\pi{\theta \over r}\right)\]

Now every time \(\theta\) changes by \(r\), the input to the sine function changes by \(2\pi\).

Let's rewrite this as:

\[\bbox[20px, border: 2px solid red]{y=\sin\left({2\pi \over r}\theta\right)}\]
